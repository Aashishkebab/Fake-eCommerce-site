List of extras outside the requirements

  • Search function that allows user to search for items that contain a specific string.
  • Sort function that allows sorting products by both price and name.
  • Media Query that changes navbar text and content sizing based on screen size.
  • Transitions and fades on many elements.
  • A price is stored in the purchase history table so that if the price changes, user can see what they paid for it.
  • Generates a unique license key (hash) for each purchase.
  • Add to cart button disabled for anything already in cart (can only purchase item once).
  • Error message if item previously purchased by that user.
  • Admin can add and delete items, but they can also change item description, title, and price.
